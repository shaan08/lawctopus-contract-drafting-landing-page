# Lawctopus Contract Drafting & Freelancing Landing Page

A modern, conversion-focused landing page designed for the Lawctopus "Mastering Contract Drafting & Freelancing" course.

## Live Demo

https://lawctopus-contract-drafting-landing.vercel.app/

## Project Objective

The objective was to design and develop a professional, responsive, and conversion-optimized landing page for legal professionals and law students interested in contract drafting and freelancing.

## Features

- Responsive design
- Bootstrap 5
- Vanilla JavaScript
- Conversion-focused UI/UX
- Accessibility support
- SEO optimization
- JSON-LD schema
- Curriculum tabs
- FAQ accordion
- Countdown timer
- Mobile navigation
- Interactive animations
- Responsive timeline
- Faculty carousel
- Pricing comparison
- Money-back guarantee section

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Accessibility

- Semantic HTML
- ARIA attributes
- Keyboard navigation
- Skip links
- Focus management
- Reduced motion support

## Performance

- Lazy loading
- Intersection Observer API
- Optimized animations
- Responsive images
