# Lawctopus Contract Drafting & Freelancing Landing Page

A modern, conversion-focused landing page designed for the Lawctopus "Mastering Contract Drafting & Freelancing" course.

## Project Objective

The objective was to design and develop a professional, responsive, and conversion-optimized landing page for legal professionals and law students interested in contract drafting and freelancing.

- Accessibility support

- FAQ accordion
- Countdown timer
- Mobile navigation
- Interactive animations

- Pricing comparison
- Money-back guarantee section
